  1. DESCRIPCION

    2. Discusion de conceptos de diseno y desarollo de paginas Web.Aplicacion de los lenguajes HTML y JavaScript conjuntamente con CSS para la creacion de paginas Web. Creacion y adaptacion de elementos de multimedios para ser utilizados en paginas interactivas. Requiere 45 horas de conferencia-laboratorio.3 creditos.

  1. OBJETIVOS

    2. Se espera que al finalizar el curso,el estudiante pueda:

    1. Discutir conceptos de diseno.
    2. Aplicar los lenguajes HTML y JavaScript conjuntamente con CSS para la creacion de paginas Web.
    3. Crear y adaptar elementos multimedios para ser utilizadas en paginas interactivas.
    4. Entender las implicaciones eticas,sociales y legales inherentes a la disciplina.
